# RTF Tables App Enhancement Summary

## Overview
The `rtf_tables_app.R` has been enhanced with three major features:
1. **Help Page** - Comprehensive user documentation
2. **PDF Loading** - Direct PDF upload and table extraction
3. **Dependency Management** - Automatic R and Python package checking

---

## 1. Help Page ("Hilfe" Tab)

### Features
- **Quick Start Guide**: 4-step guide to get started
- **Tab Explanations**: Detailed description of each tab's purpose
- **Dependencies**: Information about required and optional packages
- **Tips & Tricks**: Useful workflow recommendations
- **FAQ**: Common questions and answers

### Access
- Click the **"Hilfe"** tab (first tab in the app)
- Provides context-sensitive help without leaving the app

---

## 2. PDF Loading ("PDFs laden" Tab)

### Features
- **Multi-File Upload**: Select one or multiple PDFs
- **Automatic Extraction**: Runs `extract_tables.py` automatically
- **Status Display**: Real-time feedback on extraction progress
- **Error Handling**: Shows detailed error messages if extraction fails
- **Auto-Refresh**: File list updates after successful extraction

### How to Use
1. Go to **"PDFs laden"** tab
2. Click **"Choose File"** and select PDF file(s)
3. Click **"Tabellen extrahieren"** button
4. Monitor status in real-time
5. View extracted table count and summary
6. Switch to **"Tabellen & Vorschau"** to process extracted tables

### Requirements
- Python 3 with `pdfplumber` and `openpyxl` packages
- These are installed automatically if missing

---

## 3. Dependency Management

### Automatic Checks on Startup

#### R Packages
- Checks all required packages: shiny, readxl, flextable, officer, dplyr, stringr, DT, tools, htmltools, zip, colourpicker
- Installs missing packages automatically
- Uses `pacman` for package management

#### Python Environment
- Detects Python 3 availability
- Checks for required packages:
  - `pdfplumber` - PDF table extraction
  - `openpyxl` - Excel file generation
- Attempts automatic installation if missing
- Shows detailed status in the app

#### Optional Features
- `pdftools` and `png` - For PDF comparison feature (optional)

### Dependency Status Display
The **"PDFs laden"** tab shows:
```
✓ R-Pakete: Alle erforderlich Pakete installiert
✓ Python: Python verfügbar (version info)
✓ Python-Pakete: pdfplumber, openpyxl
✓ Optional: pdftools status
```

## Workflow Example

### Complete PDF → RTF Export
1. **Step 1**: Startup
   - App automatically checks and installs dependencies
   
2. **Step 2**: Load PDFs (Tab "PDFs laden")
   - Upload 1 or more PDFs
   - Click "Tabellen extrahieren"
   - Monitor progress in status box
   
3. **Step 3**: Select & Style (Tab "Tabellen & Vorschau")
   - Select table from list
   - Choose style from dropdown
   - Choose language (EN/DE)
   - Preview appears automatically
   
4. **Step 4**: Download
   - Single table: Click "Aktive Tabelle (RTF)"
   - Multiple tables: Go to "Batch-Export" tab, select tables, download ZIP

## Troubleshooting

### Python Not Found
- Install Python 3 from python.org or via homebrew: `brew install python3`
- Verify: `python3 --version`

### Python Packages Failed to Install
- Manual installation: `pip3 install pdfplumber openpyxl`
- Check pip: `pip3 --version`

### PDF Files Not Extracted
- Check PDF format (must be readable by pdfplumber)
- Check file permissions
- Check extract_tables.py is in working directory
- Review error messages in "Verarbeitungsstatus" box

### File List Not Refreshing
- Manually switch to "Tabellen & Vorschau" tab
- Click refresh in browser if needed

## Performance Notes

- **PDF Extraction**: Large PDFs may take several minutes
- **Status Updates**: Real-time status displayed during extraction
- **Batch Processing**: Processing multiple tables simultaneously can be memory-intensive
- **File Operations**: Temporary directories are cleaned up automatically
